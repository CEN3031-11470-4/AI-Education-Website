ARTIFICIAL INTELLIGENCE

In computer science, artificial intelligence (AI), sometimes called machine intelligence, is intelligence demonstrated by machines, in contrast to the natcural intelligence displayed by humans. Colloquially, the term "artificial intelligence" is often used to describe machines (or computers) that mimic "cognitive" functions that humans associate with the human mind such as "learning" and "problem solving."

Facebook

  • Text in posts are using deep learning to analyze the context of the language in order to prompt you to use Facebook tools, monitor hate speech, and detect suicidal intent.

  • Targeted ads are using machine learning to predict how susceptible you will be to an ad's message using profile demographics, Facebook activity, and activity on websites off of Facebook.

  • Image/video posts are using computer vision and deep learning to read your images/videos with object recognition technology to create image tags and alert for inappropriate content.

  • News Feed is using advanced machine learning to predict how interested you will be in a story and sort the order posts are seen based upon a tool called ranking.

  • Comments are using deep learning to understand how people are using slang in order to match you with advertisers, rid prohibited content, rank search results, and identify trending topics.

  • Photo tagging is using computer vision and deep learning to identify people in your photographs and make recommendations of other users to tag in your pictures.

Amazon

  • Product recommendations are using machine learning to tailor products to you base on your browsing history and purchasing data.

  • Search results are using machine learning to provide individualized product sorting to you based on your recent views, purchase history, and preferences.

  • Frequently bought together is using machine learning to provide items to you that are selected based upon data collected from purchases made by customers.

  • Pricing is using machine learning to manage the dynamic pricing model amazon uses that lowers prices when demand is low and inventory is high, and increases prices when supply is low.

  • Reviews are using machine learning to analyze the content in order to identify fake reviews from genuine ones and rank the reviews for you.

  • Delivery time is using machine learning to predict buying behavior of users and avoid stocking delays by sending products close to expected buyers in order to offer fast delivery.

TripAdvisor

  • Recommended for you is using machine learning to personalize recommendations by sorting user-generated reviews and presenting them to you based upon where the system believes you are in the trip-planning process.

  • Reviews are using machine learning to identify if a user's uploaded review will be helpful to others before publishing it and also looking for patterns in your history on the site to provide relevant, beneficial reviews to you.

  • Hero photos, the main photo of a listing, are using deep learning and computer vision to provide the most eye pleasing, relevant photo to you based upon the refinement search categories you choose.

  • Uploaded photos are using computer vision to evaluate the quality and aesthetics of photographs and notify suppliers when they upload displeasing photos.

  • Results is using deep learning to personalize result listings based upon maching you with specific interests you have provided and match listings that have reviews with those keywords.
